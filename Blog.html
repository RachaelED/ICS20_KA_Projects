First post

Posted on Tuesday, September 24, 2019

Welcome to my blog! Today I will be sharing about everything that I've been learning about on Khan Academy.

Today's lesson has focused on CSS styling properties, such as:

  • fonts
  • text size
  • line spacing
  • many other formating concepts

These concepts can all be used to style a webpage and make it look exactly as you want it.

There are certain font styles that are standard across all computers, while others are not supported by every computer.

The font styles that are standard accross all (or at least most) computers are:

  • serif
  • sans-serif
  • fantasy
  • courier
  • cursive
  • monospace

In order to ensure that the font you want shows up on all computers, you want to pick a generic font, as listed above.

That's what I learned about today!
